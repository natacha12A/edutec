Regras

Nesse nivel, você deverá associar corretamente os elementos as classificações inorgânicas correspondentes.

Nesse nivel, quando você arrastar o nome para a classificação errada, o quadrado ficará vermelho, mostrando que está incorreto.

Se errar, aparecerá uma imagem de um bichinho triste. Clique nele para ver a resposta (se quiser).

Por outro lado, se acertar, o quadrado ficará verde, indicando que está correto!

Ao finalizar o nível (valendo para todos os niveis) :  ✅ Acerte tudo e ganhe uma estrela amarela!  ❌ Se não acertar tudo, ganha uma estrela branca.
